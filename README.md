# Мир Данилов 2 — мультиплеер (без DLC)

Полноценная реализация карточной игры по правилам «Мир Данилов 2» (версия 2.0),
без DLC-карт. Каждый подключившийся игрок садится за виртуальный стол, игра
идёт по фазам (добор → действие → атака), синхронизация — через WebSocket.

## Запуск

1. Установить Node.js (если ещё нет): https://nodejs.org
2. В папке проекта выполнить:

   ```bash
   npm install
   npm start
   ```

3. Открыть в браузере: http://localhost:8080
4. Открыть эту же страницу в нескольких вкладках/устройствах в той же сети,
   ввести имя и нажать «Присоединиться» в каждой.
5. Когда все готовы (минимум 2 игрока) — любой игрок нажимает «Начать игру».
6. Каждый выбирает одного из двух предложенных боссов.
7. Играем по фазам: добор — автоматически, действие и атака — по кнопкам/картам.

## Как устроено

- `cards.js` — база всех карт (не-DLC): характеристики и текст способностей.
- `engine.js` — весь игровой движок: очередь ходов, фазы, приоритеты атаки,
  статусы (яд, вирус, оглушение, заморозка, неуязвимость, стена и т.д.),
  проверка побед/выбываний, авторитетное состояние игры.
- `server.js` — WebSocket-обвязка: принимает сообщения от клиентов, вызывает
  методы `engine.js`, рассылает каждому игроку персональное состояние (руки
  других игроков скрыты, видно только количество карт).
- `index.html` — интерфейс: стол всех игроков, рука, модалки выбора целей
  для способностей, кнопки действий.

## Что реализовано

Все белые, зелёные, синие, жёлтые (боссы) и фиолетовые карты **без DLC**
(коричневые тестовые карты по правилам не участвуют). Полностью:
фазы хода, приоритеты целей (Даны → боссы → «Приоритет 1» → остальные),
неуловимость, яд, вирус, оглушение, заморозка + бонус после неё, неуязвимость,
Стена (50%), Рост Обычного Данила, «Ускорение» каждые 5 раундов (−5 HP всем),
выбывание игроков и возврат их карт в колоды, комбо «карта + фиолетовая».

## Известные допущения (правила не давали точных цифр/порядок)

- **Количество копий карт в колоде**: не указано в правилах, поэтому взято
  по 3 копии каждой белой/зелёной/синей карты, по 2 — каждой фиолетовой,
  по 3 копии каждого босса в колоде боссов. Легко поменять в `cards.js`
  (`DECK_COUNTS`, `BOSS_COPIES`).
- **Атака «самим собой»** (когда стол пуст) доступна каждый ход по выбору
  игрока, база урона 1, с теми же правилами приоритета целей.
- **«Судья»** и другие активируемые (не пассивные) способности занимают всю
  фазу действия — в тот же ход нельзя ещё и выложить карту.
- **Множественный отряд соперников**: при атаке или эффекте, требующем
  «врага», игрок сначала выбирает конкретного соперника (если их больше
  одного), затем — карту на его столе.
- Порядок применения защиты при уроне: разовый блок Рыбака → Стена (50%) →
  вычитание из HP; отражение (Данил с длинной палкой) считается от уже
  уменьшённого урона и не влияет на урон исходной атаки.
